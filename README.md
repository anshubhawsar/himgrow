## 🏃 Quick Start

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/anshubhawsar/himgrow.git
cd himgrow

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5174`

---

## 📦 Build & Deployment

### Build for Production
```bash
npm run build
```

Output files will be in the `dist/` directory.

### Preview Production Build
```bash
npm run preview
```

### Deploy to Vercel
This project is auto-deployed on Vercel. Simply push to the `main` branch and Vercel will automatically build and deploy.

```bash
git push origin main
```
